About Me

Thank you for visiting my GitHub portfolio. Please feel free to check out my projects and leave some feedback.

You can find some of my musings within this site. You will find mostly beginer stuff in HTML, CSS and some JavaScript. Don't expect to find any advanced stuff here. The projects showcased here are ones that I have used to learn to code. Enjoy!
